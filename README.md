# VHDL Projects - CA Lab

This repository contains VHDL programs created in Xilinx for Computer Architecture Lab experiments.

## 📚 Topics Covered

### Logic Gates
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- All Gates in One Code

## 🛠 Tools Used
- VHDL
- Xilinx ISE / Vivado
- Digital Logic Design Concepts

## 📁 Folder Structure

VHDL-Projects-CA-Lab/
│── Logic-Gates/
│   ├── ALL_GATES_IN_ONE_CODE.vhd
│   ├── Basic-Logic-Gates/
│   │   ├── AND_GATE.vhd
│   │   ├── OR_GATE.vhd
│   │   └── NOT_GATE.vhd
│   ├── Universal-Logic-Gates/
│   │   ├── NAND_GATE.vhd
│   │   └── NOR_GATE.vhd
│   └── Special-Logic-Gates/
│       ├── XOR_GATE.vhd
│       └── XNOR_GATE.vhd

## 🎯 Purpose
This repository is created for learning, practice, and academic lab work.
